# 更新日志

> 这里矗立着 StudySolo 从初始 MVP (最小可行性产品) 跃升至今所有的迭代进化碑刻。每一次浴血重置皆是铺好我们征服下一片海域的坚实台阶！

## v0.2.0 (2026-03 - 引擎重构跃迁大更新)
* **后端架构涅槃突破 - 全解耦“插件化节点”实装！**：由上古的纠缠单体式业务大杂烩全面迁移至基于微子生态的运行插件系统！所有 if/else 祖传混沌长臂判定全部遭到铲除销毁，使得以后为平台研发“新的技能树节点”成本暴降至十分之一！
* **精准传导系统启动**：抛除低端粗暴的总和上下文连招，研发了依靠物理级有向无环边（Edges）实施信息的“静脉输入”，极大封印了大模型的离轨幻觉跨阶段串联。
* **业务前端代码外科手术**：实现了全站逻辑薄页面（仅路由）厚底层 Hooks（重度闭环）的最高级分层范式。
* **上帝指环超能系统（Admin Panel）实装**：十多项权限风控统辖权全部注入全景控制雷达界面之中完成封闭建设。

## v0.1.5 (2026-02 - 同步抗压与视觉力增强)
* **超限防御三级深渊防抖系统加持**：强力拦截海量拖拽下拖慢浏览器并试图对云端底层发起攻击的高频 IO，实现 0 感知极简流转不卡顿！
* **SSE 串并流长连协议深水修复**：重建并覆盖了所有 SSE 会话生命线，硬扛复杂的 Nginx 多路代理掐断挑战。
* **深夜潜行级质感（暗黑体系）覆盖**：接引 Tailwind CSS v4 的高质感表现引擎并用底层 Tokens 进行光影色彩全景变盘覆写！

## v0.1.0 远古觉醒日 (MVP 创立降诞)
* 双路逻辑决策中枢流线接通跑通（Analyzer + Planner）。
* 画布渲染器视觉与 `@xyflow` 核底的兼容融合论证并实际降落前端表面完成。
* 主生态跨项目联合 Supabase 数据库同盟、数据护栏机制初探成功并实际封测落地。
