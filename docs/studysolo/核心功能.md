# 核心功能

## 1. 动态生成式工作流 (Generative Workflow)
*   **双引擎 AI 智能拆解 (AI Analyzer & Planner)**：用户仅输入诸如“我想在三十天内入门流体力学并能看懂基础方程”的需求原话，后端的逻辑大模型会经过自我思辨（Analyzer），并自动将其在画布上生成由几十个相关依赖关系网格组装出来的“攻坚拓扑图”（Planner）。
*   **高能的交互画布 (Visual Canvas)**：基于硬件级渲染 `@xyflow/react`，将所有学习路径变为物理化的实体操作台。你甚至可以把“信息抽取”节点单独拖曳，将它的产出直接分流传递给“翻译”与“出题”两个分支节点进行并行消化。

## 2. 插件化沙盒节点与多模型引擎 (Plugin & Multi-Model Power)
*   **彻底解耦的节点插件生态**：系统的不同环节（提取大纲、网络聚合、内容结构生成、闪卡记忆测试等）均部署为完全隔离的独立微插件体系，杜绝代码缠绕。
*   **智能“代理路由”中枢**：根据任务节点所需要的不同难度与擅长方向，AI 引擎会在幕后智能委派不同的超级大脑接管。（如在逻辑推理图谱环节自动走 DeepSeek-R1 的满血强化思考，在需要格式化语气的环节智能降级至高性价模型），且内置秒级灾备切换防止平台偶发中断。
*   **拓扑按序流式引擎 (SSE Streaming)**：不是一问一答，而是引擎沿着连线拓扑流传！你能像看着工厂流水线一样看着上游的分析产物如同液体一般通过连线流入下游进行组装。

## 3. 三层防抖追踪与超稳态同步 (Sync Engine)
*   为了防止高频次且庞杂的图谱拉扯操作拖断云端数据库，独家采用了 `UI (无感知 0ms) -> IndexedDB (前端数据暂存 500ms) -> Supabase (静默云网同步 4s)` 的三级降压落盘策略，在极速操作下保证云记录的一致与安全永不丢包。
*   **One-ID 深层互通**：1037Solo 主平台的所有用户资产（Quotas，权益，个人标记）都可以与 StudySolo 完全等价流转。

## 4. 全景监控的后台超视距指挥塔 (Admin Dashboard)
*   内建了庞大且细致至极的管理员后方阵地，具备十大专属管理维度空间，可以上帝视角实时调度所有算力接口并发并进行风险风控审计。
