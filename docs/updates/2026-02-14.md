#### v2.3.091 📊 AI 成本预测模块 + Tailwind 体系重建（含重大架构变更）
这是一次涉及**构建配置重建**的重要版本，同时新增了 AI 模型成本预测功能。

**构建配置重建（3 个配置文件）：**

| 新增文件 | 职责 |
|---------|------|
| `tailwind.config.js` | Tailwind CSS 配置文件（此版本首次加入，说明之前通过其他方式引入或重新规范化配置） |
| `postcss.config.js` | PostCSS 处理器配置（Tailwind 所需的转换管道） |
| `index.css` | 全局样式入口（引入 Tailwind 指令 `@tailwind base/components/utilities`） |

> ⚠️ 注意：这三个文件在 v090 中不存在，但在 v091 中出现，表明该版本进行了 CSS 工具链的重新规范化，可能是从内联 Tailwind CDN 方式迁移到本地 PostCSS 构建链。

**AI 成本预测子模块（12 个新文件）：**

这是一个完整的 AI 模型使用量预测与可视化模块，位于 `components/features/admin/prediction/`：

| 新增文件 | 职责 |
|---------|------|
| `prediction/ModelCharts.tsx` | 使用图表组件可视化不同模型的历史使用量与成本趋势 |
| `prediction/ModelInputPanel.tsx` | 预测参数输入面板：输入日均查询量、模型分布比例等 |
| `prediction/ModelResultPanel.tsx` | 预测结果展示面板：显示预计月成本、各模型分摊费用 |
| `prediction/ModelTable.tsx` | 模型价格参数表格（单位 Token 成本、上下文长度、定价等级） |
| `pages/admin/CostModelPage.tsx` | 成本预测完整页面（路由入口） |
| `pages/admin/DocsPredictionPage.tsx` | 文档型预测页面（用于预测文档分析类 AI 调用成本） |
| `constants/prediction.ts` | 各模型单价常量（Tier 1/Tier 2 各模型的 Input/Output Token 价格表） |
| `types/prediction.ts` | 预测相关 TypeScript 类型定义（PredictionInput、PredictionResult 等） |
| `utils/prediction.ts` | 成本计算核心逻辑（Token 估算、多模型加权成本公式） |
