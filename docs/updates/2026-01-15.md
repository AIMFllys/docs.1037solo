### 【面向开发者】

#### v2.2.045 🐛 服务端玄学问题排查
- 深入排查部分最新版模型（Qwen / DeepSeek 最新版）API 调用失败的玄学问题。
- 增加 API 请求重试逻辑与降级回退机制。

#### v2.2.046 🔒 安全加固
- 加强后端 Token 校验逻辑，防止伪造请求绕过配额限制。
- 完善速率限制（Rate Limiting）中间件配置。

#### v2.2.047 🗃️ 数据库优化
- 优化 MySQL 会话查询索引，减少慢查询。
- 修复会话归属校验漏洞（用户 A 可读取用户 B 会话的 Bug）。

#### v2.2.048 📤 文件上传功能
- 实现前端文件拾取与 FormData 上传。
- 后端实现文件接收、存储与内容解析（支持 PDF、TXT、图片等）。
- 最大文件限制：20MB，完成前端 size 校验与后端双重验证。

#### v2.2.049 🧪 KaTeX 渲染兼容性修复
- 修复部分 LaTeX 表达式（如 `\begin{cases}`）渲染出错的问题。
- 解决 KaTeX 与 Markdown 混排时的转义冲突。

#### v2.2.050 🎨 代码块样式修复
- 修复代码块在深色模式下对比度不足的问题。
- 统一代码块字体为 `JetBrains Mono`。

#### v2.2.051 🔢 数学公式渲染完整集成
- 完整集成 `remark-math` + `rehype-katex` 管道。
- 测试验证行内公式（`$...$`）与块级公式（`$$...$$`）均可正确渲染。

#### v2.2.052 🚀 流式响应性能优化
- 优化 SSE 流式数据到 React 状态的更新策略，减少不必要的整页重渲染。
- 修复长回答超过 4096 Token 时流式截断的 Bug。

#### v2.2.053 📧 邮件验证系统修复
- 修复注册邮件发送失败的 SMTP 配置问题。
- 优化邮件模板样式，改进可读性。

#### v2.2.054 📱 移动端专项优化
- 重构移动端底部导航栏组件，避免与系统底栏重叠。
