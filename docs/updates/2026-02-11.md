#### v2.3.084 🔍 Perplexify 深度搜索工作流子系统
这是一次架构级的功能新增，完整实现了三段式 AI 搜索工作流（类 Perplexity/DeepSeek 风格）：

**新增文件（11 个代码文件）：**

| 文件 | 职责 |
|------|------|
| `perplexify/WorkflowContainer.tsx` | 工作流容器，过滤 pass3（最终回答）步骤，只显示推理过程步骤 |
| `perplexify/WorkflowStepItem.tsx` | 单个工作流步骤卡片，含 pending/streaming/completed/error 四种状态 |
| `perplexify/WorkflowErrorBoundary.tsx` | 工作流级错误边界，防止单步骤崩溃影响整体 |
| `perplexify/QuotePanel.tsx` | 来源引用面板，展示 AI 引用的网页来源卡片列表 |
| `perplexify/InlineCitation.tsx` | 行内角标引用组件，点击可跳转到 QuotePanel 对应来源 |
| `perplexify/SourceButton.tsx` | 来源按钮，含网站 Favicon + 标题的紧凑展示 |
| `hooks/usePerplexifyWorkflow.ts` | 管理三段式工作流状态机（pass1 查询分析 → pass2 搜索 → pass3 总结） |
| `hooks/useQuotePanel.ts` | 控制 QuotePanel 的展开/收起状态和引用跳转逻辑 |
| `hooks/useQuoteParser.ts` | 解析 AI 回复中的 `[1]`、`[2]` 等引用标记，映射到来源 URL |
| `constants/perplexifyPrompts.ts` | 三段式提示词常量（System Prompt for 查询分析/联网搜索/综合总结） |
| `types/perplexify.ts` | WorkflowStep、WorkflowStatus 等 TypeScript 类型定义 |
| `components/ui/SoloWave.tsx` | 新增 SoloWave 动效组件（AI 思考中的波浪动画） |

**核心架构说明：**
```
WorkflowContainer
  └── WorkflowStepItem × N  (steps: pass1, pass2)
        └── WorkflowErrorBoundary (单步错误隔离)
  └── [pass3 被过滤，用于主界面最终回答]
QuotePanel
  └── SourceButton × N
InlineCitation (嵌入在 Markdown 渲染内)
```

- **文件规模**：464 个文件（+7 TSX, +6 TS vs v080）。
