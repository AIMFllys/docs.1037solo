# ⚡ 核心功能

> 1037 SOLO 是专为华中科技大学（HUST）学子打造的下一代 AI 智能助手与学术探索平台，集成全球顶尖 AI 模型，深度融合校内服务与学术资源。

---

## 平台定位

"**1037**" 象征着华中科技大学的文化代码，代表严谨的学术氛围与创新精神；"**SOLO**" 寓意着独立思考、深度探索的个人化学习体验。

我们致力于解决学术研究中的五大痛点：

| 痛点 | 解决方案 |
| :--- | :--- |
| 信息碎片化与检索效率低 | 聚合搜索 + AI 智能摘要，告别广告干扰 |
| AI 工具分散 | 多模型一站式聚合，无需逐个切换账号 |
| 学术资源获取门槛高 | 统一导航 + 分类推荐，一键直达 |
| 个性化知识管理缺失 | 对话历史 + 书签 + 文件夹系统 |
| 使用成本高昂 | 灵活的配额管理 + 兑换码机制 |

## 一、智能搜索与探索

将 AI 的分析能力与传统搜索引擎的广度完美结合，为您提供**纯净、无广告**的搜索体验。

### 聚合搜索

- 集成主流搜索引擎与学术数据库
- 自动过滤广告，优先展示 `.edu`、GitHub、Wiki、知网等高质量结果
- 来源去重：智能合并重复链接
- 域名识别：自动为学校官网、GitHub、PDF 文档标记专属标识

### AI 智能摘要

- 对搜索结果进行 AI 实时分析与总结
- 直接给出核心回答，而非单纯的链接列表
- 支持切换不同 AI 模型来分析搜索结果
- 可在摘要下方继续追问

### 资源导航（1037Explore）

- 精选学术资源、效率工具、常用网站导航
- HUST 官方平台快捷入口（Hub 系统、研究生院、教务处、图书馆等）
- 学术科研资源库（IEEE、Nature、ArXiv、CNKI 等）
- 支持按学科分类和访问权限筛选

## 二、多模型 AI 对话

接入全球数十种最强 AI 模型，提供媲美真人的深度对话体验。

### 模型矩阵

**国产旗舰模型：**

| 模型 | 特点 |
| :--- | :--- |
| DeepSeek V3 | 综合能力极强，日常助手首选 |
| DeepSeek R1 | 深度推理专家，擅长数学、代码、复杂逻辑 |
| 通义千问 Max | 文本生成能力卓越 |
| 通义千问 Plus（深度思考） | 平衡速度与深度 |
| Kimi K2 | 超长文本分析专家 |
| GLM-4.7 | 综合素质均衡，超大输出 |
| 1037 HUSTomni | 华科专属智能路由，更懂校园生活 |

**国际顶尖模型：**

| 模型 | 特点 |
| :--- | :--- |
| GPT-4 Turbo / GPT-5 | 高质量通用 / 旗舰能力 |
| Gemini 2.5 Pro | 超长上下文（1M+ tokens），多模态 |
| Grok 4 | 实时信息处理，搜索增强 |

### 核心能力

- **深度思考（DeepThink）**：开启后模型先进行思维链推理（CoT），适合解题、写代码、复杂策划
- **联网搜索**：实时访问互联网获取最新资讯，附带引用来源
- **文件分析**：上传 PDF、Word、TXT 文件，进行论文速读、简历润色
- **临时对话**：无痕模式，关闭窗口即焚，保护隐私
- **Perplexify 模式**：学术级深度检索，输出结构化的研究报告与参考文献
- **流式响应**：打字机效果实时输出，交互体验流畅

### 角色扮演系统

| 角色 | 风格 | 适用场景 |
| :--- | :--- | :--- |
| 🔵 深度教学 | 随和智慧的学长，循序渐进讲解 | 系统学习新知识、深入理解概念 |
| 🟢 引导启发 | 严谨的辅导专家，不直接给答案 | 作业辅导、解题训练、培养独立思考 |
| 🟡 复习评估 | 资深评估专家，出题并生成报告 | 考前复习、知识巩固、自测评估 |

## 三、个性化账户体系

### 用户权益管理

- 基于配额（Quota）的权益体系
- 实时查看各模型今日剩余调用次数
- 额度每日 00:00 自动重置
- 兑换码系统：通过兑换码获取额外使用加成或 VIP 权限

### 数据同步与管理

- 对话历史云端保存，支持多端同步
- 书签管理：导入 HTML 书签文件或同步 Chrome 书签
- 文件夹系统：多级嵌套分类管理对话
- 归档管理：不常用的对话可归档，保持工作区整洁

### 个性化设置

- **主题切换**：深色 / 浅色 / 跟随系统
- **多语言**：中文、English、日本語
- **动效控制**：流光边框、悬浮 Logo、动态膨胀、开屏动效均可独立开关
- **输入模式**：极速模式（Enter 发送）/ 防误触模式（Ctrl+Enter 发送）
- **AI 页面布局**：自定义功能按钮的显示位置

## 四、全平台适配

### 桌面端

- 左侧侧边栏导航：对话记录、书签、探索中心等快速入口
- 右侧目录导航：当前文档的章节目录
- 完整的键盘快捷键支持

### 移动端

- 专属底部五键导航栏，配有聚光灯跟随效果
- AI 悬浮球：搜索页面快捷查看 AI 摘要
- 侧边栏自动折叠为抽屉菜单
- AI 对话时底部导航栏自动隐藏，最大化使用空间
