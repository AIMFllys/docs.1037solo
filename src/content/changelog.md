## v2.3.099 (2026-02-24) - Current

### 1. 平台文档与规划体系全面建立 (v2.3.098)
- **项目文档体系重建**: 建立了完整的项目文档与规划体系，参考 StudySolo 的 `Plans/ + Vibe Coding/ + Updates/` 结构
- **新增文档**: 包括根目录 README.md、docs/README.md、docs/Plans/README.md、docs/Plans/PROJECT_PLAN.md 等多个文档文件
- **文档归档调整**: 将数据库迁移文档迁移至 `docs/Plans/accumulate_plan/`，同步更新相关引用路径

### 2. 平台架构全面升级 (v2.3.094)
- **底层代码结构优化**: 进行了全面优化重组，采用业界最新的前端架构规范
- **稳定性提升**: 大幅提升了平台的稳定性、加载速度和未来功能扩展能力

### 3. 前后端认证机制深度重构 (v2.3.095)
- **域名迁移**: 后端 API 迁移至 `main.api.1037solo.com`，前端迁至 `platform.1037solo.com`
- **混合架构支持**: 适配 `platform.1037solo.com/home` 等子路径部署模式，确保认证状态跨子应用无缝共享
- **部署指南**: 新增了详尽的部署指南，涵盖 DNS 解析、宝塔配置及 SSL 证书
- **数据库预留**: 抽象数据库层配置，为未来平滑迁移至 Supabase 做好架构储备

### 4. 部署与安全优化
- **子路径部署修复**: 针对 `/home/` 子路径部署，优化了 Vite 构建配置，确保资源正确加载 (v2.3.096)
- **HTTPS 安全修复**: 将页面中残留的 `http` 协议图片链接统一升级为 `https`，消除安全警告 (v2.3.097)

### 5. 问题修复
- **v2.3.095 - v2.3.099**: 修复了若干已知问题，提升平台稳定性和用户体验
